# shortie-cli
The CLI companion to shortie.sh

Download from releases

usage:
```
This is the command line companion for shortie.sh.
        shortie is a URL shortener that is similiar in featureset to bit.ly or tinyurl. 
        The main differences are that it is much simpler to use, open-source, and privacy respecting!

        The source code is available at https://github.com/shortie-sh/server
        The source code for this CLI is available at https://github.com/shortie-sh/cli

Usage:
  shortie [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  get         Get existing redirect
  help        Help about any command
  new         Create new redirect

Flags:
  -h, --help              help for shortie
      --instance string   shortie-server instance (default "https://www.shortie.sh")

Use "shortie [command] --help" for more information about a command.
```
